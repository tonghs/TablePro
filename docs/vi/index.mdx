# TablePro

Ứng dụng database client macOS gốc. Nhanh, nhẹ, tích hợp AI SQL. Xây dựng trên SwiftUI và AppKit, sử dụng framework gốc của Apple để đạt hiệu suất thực sự trên macOS.

## Tính năng chính

### Trình soạn thảo SQL
Tô sáng cú pháp, số dòng, và autocomplete tự thích ứng theo schema database.

<Tip>
Sử dụng `Cmd+Enter` để thực thi câu truy vấn hiện tại hoặc đoạn text đã chọn.
</Tip>

### Bảng dữ liệu
Bảng hiệu suất cao, dạng spreadsheet. Hỗ trợ chỉnh sửa inline, sắp xếp và lọc.

### Cấu trúc bảng
Xem và sửa cột, index, khóa ngoại từ giao diện trực quan.

### Nhập & Xuất
Nhập từ CSV, JSON và file SQL. Xuất ở nhiều định dạng để backup hoặc migration.

### Lịch sử truy vấn
Tìm kiếm toàn văn trong toàn bộ lịch sử truy vấn. Chạy lại hoặc sửa query cũ ngay lập tức.

### AI SQL Assistant
Viết query bằng ngôn ngữ tự nhiên, nhận gợi ý inline kiểu Copilot, click chuột phải để giải thích hoặc tối ưu SQL. Hỗ trợ Claude, OpenAI, Gemini, OpenRouter, Ollama và endpoint tùy chỉnh.

## Cơ sở dữ liệu được hỗ trợ

| Cơ sở dữ liệu | Trạng thái | Cổng mặc định |
|----------|--------|--------------|
| MySQL | Hỗ trợ đầy đủ | 3306 |
| MariaDB | Hỗ trợ đầy đủ | 3306 |
| PostgreSQL | Hỗ trợ đầy đủ | 5432 |
| SQLite | Hỗ trợ đầy đủ | Không áp dụng (dựa trên file) |
| Amazon Redshift | Hỗ trợ đầy đủ | 5439 |
| MongoDB | Hỗ trợ đầy đủ | 27017 |

## Yêu cầu hệ thống

- **macOS**: 14.0 (Sonoma) trở lên
- **Kiến trúc**: Apple Silicon (arm64) hoặc Intel (x86_64)
- **Dung lượng**: ~50 MB cho ứng dụng
